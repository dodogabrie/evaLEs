# evaLEs
Evaluate the Lyapunov Spectrum of a dynamical system described by ODEs 

Project inspirated from [paulrozdeba/pyLyapunov](https://github.com/paulrozdeba/pyLyapunov). 

# How to install:
```
pip install evaLEs
```
